Testimonials

Teinda Online

Desarrollé una tienda en línea simple que permite a los usuarios navegar y comprar alimentos asiáticos. El frontend utiliza HTML, CSS y JavaScript, mientras que el backend está basado en Flask y Python, y los datos se almacenan en una base de datos MySQL. Este proyecto me permitió mejorar mis habilidades básicas en desarrollo full stack.

Restaurante

Desarrollé un sistema de gestión de restaurantes que permite a los usuarios navegar por el menú, realizar pedidos y gestionar reservas. El frontend utiliza HTML, CSS y JavaScript, mientras que el backend está desarrollado en PHP, montado en un entorno XAMPP, y la gestión de datos se realiza a través de phpMyAdmin. Aunque actualmente el proyecto está en pausa, esta experiencia me ayudó a mejorar mis habilidades en desarrollo frontend y backend, así como en gestión de bases de datos.
